S5. Model comparison

This section reports model comparison results used to identify the best-supported model for each analysis.

All candidate models were evaluated using leave-one-out cross-validation (LOO-CV), which estimates out-of-sample predictive performance. Models are ranked by the difference in expected log predictive density (elpd_diff) relative to the top-ranked model.

  • The best-supported model in each table has elpd_diff = 0.

  • se_diff gives the standard error of the difference in ELPD, indicating uncertainty in the ranking # Models syntax

All candidate models were evaluated using leave-one-out cross-validation (LOO-CV), which estimates out-of-sample predictive performance. Models are ranked by the difference in expected log predictive density (elpd_diff) relative to the top-ranked model.

- The best-supported model in each table has `elpd_diff = 0`.

- `se_diff` gives the standard error of the difference in ELPD, indicating uncertainty in the ranking
